5Euclidean vector spaces

Big data are made up of many numbers in data sets. Such data sets can be represented as vectors in a high dimensional euclidean vector space. A vector is nothing but a list of numbers, but we need to talk mathematically about the size of a vector and perform operations on vectors. The purpose of this chapter is to set the stage for this, especially by introducing the dot product (or inner product) for vectors. Having a dot product is immensely useful and we give several applications like linear regression and the perceptron learning algorithm
In the last part of the chapter rudimentary basics of analysis are introduced like sequences, continuous functions, open, closed and compact subsets. Some results will in this context only be quoted and not proved.
